ii>CharacterStream:-
The character stream uses the byte stream to perform the physical I/O, while
the character stream handles translation between characters and bytes. FileReader,
for example, uses FileInputStream , while FileWriter uses FileOutputStream .
with the help of this stream we perform the operation on the file.
CharacterStream classes are used to work with 16-bit Unicode characters. They can 
perform operations on characters, char arrays and Strings. However, the CharacterStream
classes are mainly used to read characters from the source and write them to the
destination.
CharactorStrem:-FileReader(Read file data) & FileWriter(Write file data).
